# Jinder - Job Application Tracker

Jinder is a comprehensive job application tracking system that helps you manage and organize your job search process.

## What is Jinder?

Jinder is a job application tracker, not a dating app! It's designed to help job seekers:
- Track job applications across multiple platforms
- Monitor application status and progress
- Store important details about positions and companies
- Manage interview schedules and follow-ups
- Analyze job search metrics and success rates

## Features

- **Application Management**: Keep track of all your job applications in one place
- **Status Tracking**: Monitor where each application stands in the process
- **Company Profiles**: Store information about companies you're interested in
- **Interview Scheduling**: Manage upcoming interviews and important dates
- **Analytics**: Get insights into your job search performance

## Tech Stack

- Frontend: React.js
- Backend: Node.js/Express
- Database: MongoDB
- Authentication: JWT

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Set up environment variables
4. Run the development server: `npm run dev`
